Система для ведения каталога жанров видеокассет и видеодисков, с возможностью проката и отслеживанием информации о клиентах (кто, когда и до какого числа взял фильм). База данных спроектирована в SQLite, включает 8 таблиц. Интерфейс реализован на C# с .NET Framework включает формы для ввода данных, запросы к БД и отчеты. Система обеспечивает контроль статуса носителей, расчет стоимости проката и управление возвратами.
